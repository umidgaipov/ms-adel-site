# Сайт портфолио Адели - Бьюти услуги

Личный веб-сайт для представления бьюти-услуг и портфолио мастера Адели.

## 📁 Структура проекта

```
lash site/
├── index.html           # Главный файл сайта
├── css/
│   └── styles.css      # Основные стили (пастельная палитра)
├── js/
│   └── script.js       # Интерактивность и функциональность
├── assets/
│   └── images/         # Папка для фотографий и изображений
├── docs/               # Документация и заметки
└── README.md          # Этот файл
```

## 💡 Как начать

1. Открыть `index.html` в браузере
2. Добавить фотографии в папку `assets/images/`
3. Отредактировать содержимое в `index.html` (текст, ссылки)
4. Отрегулировать стили в `css/styles.css` под свои нужды
